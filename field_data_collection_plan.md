# 롯데웰푸드 현장 데이터 수집 계획

## 🎯 **데이터 수집 목표**

### **정량적 목표**
- **정상 데이터**: 100시간 이상 (20개 냉동고 × 5시간)
- **이상 데이터**: 50시간 이상 (고장 유형별 10시간)
- **총 데이터량**: 150시간 (약 900GB)
- **수집 기간**: 2주

### **정성적 목표**
- 실제 현장 환경의 소음 패턴 파악
- 고장 유형별 특성 분석
- 환경 요인(온도, 부하) 영향 분석

### **1단계: 정상 냉동고 설치 (1주차)**
```

├── 소형 냉동고 3대 (정상 작동)

├── 수집 시간: 24시간 연속

🏪 롯데웰푸드 대리점 B

```

### **2단계: 이상 냉동고 설치 (2주차)**
```
🔧 고장/이상 냉동고
├── 베어링 마모: 1대

├── 냉매 누출: 1대 (가능한 경우)

```
